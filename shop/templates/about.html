Django E commerce Website

Welcome to Django E-commerce website. This website is built using Django framework, Python, HTML, CSS and JavaScript. It provides a user-friendly interface for shopping, checkout, and order management.

The website features a registration and login system, securely storing user information using Django's built-in authentication framework. Users can create accounts, log in, and view their orders.

The website also includes a product catalog, allowing users to browse and add products to their cart. Users can apply filters and sort products based on various criteria.

To place an order, users must log in and select products from the cart. They can then review their order, update their shipping address, and choose a payment method. After submitting the order, the website will process the payment and send an email confirmation to the user.

The website also includes a shopping cart and a checkout page
